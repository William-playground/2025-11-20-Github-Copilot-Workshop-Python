# Pomodoro Timer

ポモドーロ・テクニックを実践するためのシンプルなタイマーアプリケーション

## 機能 (Features)

### タイマー状態管理
- **作業中 (Working)**: 緑色の円形プログレス、デフォルト25分
- **休憩中 (Break)**: 青色の円形プログレス、デフォルト5分
- **停止 (Stopped)**: グレーの円形プログレス、準備完了状態
- **一時停止 (Paused)**: 現在の状態を保持し、再開可能

### 円グラフの動的描画
- SVGベースの円形プログレスリング
- 状態に応じた色分け（作業中: 緑、休憩中: 青、停止: グレー）
- 作業中はパルスアニメーション

### コントロール
- 開始ボタン: タイマーを開始
- 一時停止ボタン: タイマーを一時停止
- リセットボタン: タイマーをリセット

### カスタマイズ可能な設定
- 作業時間の長さ（1〜60分）
- 休憩時間の長さ（1〜30分）

## 使い方 (Usage)

1. リポジトリをクローン

`pomodoro.html` をブラウザで直接開くこともできます。

## ポモドーロ・テクニックとは

ポモドーロ・テクニックは時間管理術の一つで、短い休憩を挟みながら集中して作業を行う手法です。

1. 25分間作業に集中（作業セッション）
2. 5分間休憩
3. これを繰り返す

## 技術スタック (Tech Stack)

- **HTML5**: 構造
- **CSS3**: スタイリングとアニメーション
- **JavaScript (ES6+)**: タイマーロジックと状態管理
- **SVG**: 円形プログレスバー
